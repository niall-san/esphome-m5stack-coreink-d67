# 2026-06-25 001 Baseline Scaffold

## Goal

Create the local-first repository and initial ESPHome external component for
the confirmed M5Stack CoreInk `0154BN-D67-D2` panel.

## Hypothesis

The physical panel behaves like a Good Display `GDEY0154D67` / `SSD1681`
1.54 inch 200x200 SPI e-paper display. A dedicated component should remove the
YAML color inversion required by the current `waveshare_epaper` `1.54inv2`
workaround.

## Exact Config And Code State

Repository scaffolded as:

```text
m5stack-coreink-d67-esphome/
  components/m5stack_coreink_d67/

  docs/
  ledger/
```

Initial component:

- Platform: `m5stack_coreink_d67`
- Display size: `200x200`
- Buffer: 1 bit per pixel, `5000` bytes
- Default DC: `GPIO15`
- Default BUSY: `GPIO4`
- Default power hold: `GPIO12`
- Reset pin: optional, omitted in examples
- Explicit CS in examples: `GPIO9`
- SPI CLK/MOSI in examples: `GPIO18`/`GPIO23`
